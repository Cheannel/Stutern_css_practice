CSS Practice and Importance

The meaning and the importance of CSS are stated below

The term CSS means Cascading Style Sheet. CSS is a type of styling language. CSS is responsible for displaying and designing the layout of a website, which can be used alongside with Html. The CSS describes how the Html elements should be displayed.

THE CSS SYNTAX: The CSS syntax involves a rule that consists of a selector and a declaration block.

  1. Selector: The selector is used to point to the Html element that is to be styled.
  2. Declaration block: The declaration block is surrounded by curly brackets. It can contain one or more CSS property name and value, separated by a colon.
    • Property: Property is a type of attribute of an html element. Examples; margin, color, fonts, etc.
    • Values: Values are attributes assigned to the CSS properties. Examples; red, italic, center, bold, etc.

    METHOD OF CSS: The method or type of CSS includes the various ways a website layout can be designed. It is further explained using examples

    1. In-Line CSS: This styling method involves using the CSS styling at the opening tag of an HTML element as a style attribute.
    2. Internal CSS: It is a styling method used only on a single HTML page. This method involves declaring the CSS styling in the "head" tag section of an HTML file, this is done after the "title" tag, within an "style" element.
    3. External CSS: This method can be defined using an external file. This file mustn't contain any HTML code, and it is saved with a ".css extension". It is a styling method that can be used on more than one HTML page. To use an external style sheet, add a link to it in the tag section of each HTML page. The path of the external file is defined in the " href attribute".

    CSS SELECTORS

    1. CSS Class Selectors: The CSS Class selector is declared using a dot(.) positioned in front of the name of the class. The class name is defined by the developer.
    2. CSS ID Selector: The CSS ID Selector is used by a developer to create ids and matches them to their styling contents. This can be done using the hash (#) symbol.
    3. CSS Element Selector: The CSS element selector is used to match the HTML elements having the same name. A selector of <"h1"> matches all the <"h1"> elements. Examples of other element selectors include, <"body">, <"li">, <"ol">, <"table">, <"p">, etc.
    4. Universal CSS Selector: The Universal CSS selector is used to select all the elements in an HTML file. This is declared using the asterisk (*) symbol positioned at the beginning of the curly braces.
    5. CSS Attribute Selector: The CSS Attribute selector styles content according to the attribute and the attribute values mentioned in the square brackets.

    CSS COMMENTS Comments are used to explain the code, and may help when you edit the source code at a later date. CSS comments are not displayed in the browser, they can help in documenting a code's source. It is also useful in helping other users make use of the code. A CSS comment is placed inside the "style" element, and starts with /* and ends with */.

    THE CSS BOX MODEL The CSS box model refers to how HTML elements are modelled in browser engines and how the dimensions of those HTML elements are derived from CSS properties. It is a fundamental concept for the composition of HTML webpages.

    IMPORTANCE OF CSS

    1. CSS are the basis of how an Html page is displayed, so therefore, it helps In designing an Html layout.

Difference Between Padding, Margins and Borders

Padding Margin Borders
Paddings can be defined as the spaces between the element's contents and the border. Margins involves the spaces that surrounds the contents. Borders can be defined as lines that act as the edges that surrounds an element.
Paddings can not be designed with a variety colors. Margin can also not be designed with a variety colors, they are just mere spaces around the border. Borders can be designed with a variety colors.
Paddings do not get styled differently, unlike the border box. Margins also can not be styled differently. Borders are styled differntly and can be beautified.
